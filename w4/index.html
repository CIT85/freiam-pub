Any Feedback?

httpbin is a site that can be used to see the response of a form such as this in a test environment. Within this environment, the submitted responses are seen in alphabetical order, rather than their order on the form itself, and any that are not responded to are left blank by the site. Submitting will link to the site with the inputs detectable in the site's generated url, where as posting it will not, making it more secure in the scenario where personal information is being submitted.
