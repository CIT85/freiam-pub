What Do I Wish to Learn With HTML?

All tags and definitions courtesy of W3 School and their HTML Reference by Alphabet page.

The <noscript> tag

  • This tag is an alternate definition flag with the purpose of only being displayed for those who do not have scripts enabled or supported in their current browser. With this, it's a way to not completely alienate those unable to see script specific information in a given webpage, something I wouldn't want to have occur.

The <select> and <option> tags

  • These tags work alongside each other as a way to create a list of options for the sake of processing the user's personal input, making it very critical for any site that would have a use for user input, such as needing to swap a specified language or color requirement. The input and subsequent response from these tags are useful for both being able to accommodate others and be capable to change differing options for the page's contents, such as changing the colors of a car model.

As I progress through this class, I look forward to fully understanding how to utilize these aforementioned tags and the many others this programming language offers.
